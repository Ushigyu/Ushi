ゲーム名：Treasure Collect
           (ファイル名はTreasure Cllectorになっていますが、本当
　　　　　　のタイトルはTreasure Cllectです）
使用エンジン：unity
使用バージョン：unity 2022.3.23f1

ルール
球体を操作し、マップにある計4つの黄色の球体をすべて取ることが出来たらクリアです。黄色の球体に当たることで自動的に残りお宝が1減ります。
赤黒い色の棒や壁、床に触れるとゲームオーバーになります。

操作方法
・タイトル：画面真ん中にあるボタンにマウスカーソルをあわせてクリックするとゲームスタート
・ゲームシーン：W,A,S,Dで移動
・ゲームオーバーシーン：画面真ん中に２つのボタンがあり、スタートシーン同様、マウスカーソルをあわせてクリックすればいい。左のボタンはタイトルに戻る。右のボタンはもう一度ゲームを始めることが出来る。
・クリアシーン：画面真ん中にあるボタンにマウスカーソルをあわせてクリックするとタイトルに戻る。

アピールポイント
簡単なルールと簡単なゲーム性、単純な操作方法で誰でもできるゲームでありながら、障害物の間を通る時のハラハラ感と集中が売りです。ステージ後半にある、動く棒や壁のギミックは、通りずらいが規則性があり、一回で行くのは難しいが何度も挑戦してタイミングをつかみ、やっとの思いでクリアできた時の達成感もこのゲームのアピールポイントの一つです。
